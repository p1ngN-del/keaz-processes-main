Стандарт 37: Предоставление информации к запросу на подбор кодов ТН ВЭД

КЭАЗ · Обязателен к исполнению всеми инициаторами запросов
📋 Обязательный состав запроса
  1. Цель ввоза/вывоза — контрактная поставка / неконтрактная поставка / платная / бесплатная / выставочный образец (с возвратом / без возврата) / рекламация / ремонт и т.п.
  2. Направление поставки — экспорт / импорт / временный ввоз / временный вывоз.
  3. Таблица Excel — артикул + наименование из 1С:УПП (артикулы должны соответствовать тех.документации).
  4. НS Code изготовителя (предпочтительно) или поставщика, который будет указан в экспортных декларациях страны отправления.
  5. ФИО технического специалиста (конструктор, технолог, продакт-менеджер и т.п.) для консультаций по возникающим вопросам.
⚠️ ВАЖНО: Указанный перечень не является исчерпывающим. Специалист по декларированию обязан получить всю информацию для однозначной идентификации кодов. Инициатор обязан её предоставить.
❗ ТРЕБОВАНИЕ К ДОКУМЕНТАЦИИ: Вся документация от имени ГК КЭАЗ для таможенных органов должна быть на официальном бланке, иметь подпись составителя и контактный телефон. Ответственность за достоверность информации лежит на инициаторе проработки.
🌐 Сайт производителя

Необходим для изучения товара, когда предоставленной информации недостаточно. Особенно важно при наличии объектов интеллектуальной собственности в реестре ФТС с ограничением ввоза. Также предоставляется при дополнительных запросах ФТС.
